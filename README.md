## Install

Clone this repository  
`git clone git@github.com:axsmak/memeat.git`  
or  
`git clone https://github.com/axsmak/memeat.git`  

Enter to the folder  
`cd memeat`

And run script from superuser  
`sudo ./install`

## Uninstall

Enter to the folder  
`cd memeat`

And run script with superuser  
`sudo ./remove`
